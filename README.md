This project explores three decades of Formula One racing data (1989â€“present) to uncover patterns in driver performance, constructor dominance, race-time variability, and circuit behavior. Using Python, SQL, and statistical modeling, the analysis cleans and transforms a unified dataset before generating insights on qualifying vs. final results, lap time distributions, constructor competitiveness, and track-level variability.
